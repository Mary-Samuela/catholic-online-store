# 🛍️ E-Commerce Web Application

## 📌 Project Overview

This project is a full-stack e-commerce web application developed to demonstrate practical skills in modern web development. It includes a responsive frontend, a secure backend API, and full deployment, providing a complete online shopping experience.

## 🎯 Features

- User authentication and authorization
- Product browsing and management
- Shopping cart functionality
- Admin dashboard for product control
- Secure backend with API integration
- Fully deployed and accessible online

## 🧱 Project Structure

### **Step 1 — Frontend UI (React Router v7 + Tailwind CSS)**

- **Layout & Navigation**: Structured layout with responsive navigation
- **Home Page**: Landing page showcasing key content
- **Products & Shop**: Display of products with browsing functionality
- **Cart, Authentication & Admin**:
  - Add/remove items from cart
  - User login/register
  - Admin interface for managing products

### **Step 2 — Backend API (Node.js + Express + MongoDB)**

- **Database Models**: Structured schemas for users, products, and cart
- **REST API Routes**: Endpoints for CRUD operations
- **Authentication (JWT)**: Secure login and protected routes
- **Payments**: Integration for handling transactions

### **Step 3 — Frontend & Backend Integration**

- Connected using **Fetch**
- Dynamic data flow between client and server
- Real-time updates for cart and products

### **Step 4 — Deployment**

- Frontend and backend deployed on cloud platforms
- Database hosted on MongoDB Atlas
- Configured environment variables for production

## 🛠️ Technologies Used

- **Frontend**: React, React Router v7, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JSON Web Tokens (JWT)
- **Deployment**: Vercel

## 🚀 Getting Started

### Prerequisites

- Node.js installed
- MongoDB ( Atlas)
